
## Description

[ReliefVideoApp](https://github.com/GuilhermeGabriel/ReliefVideoAppFrontEnd) Backend for ReliefVideoAppFrontend

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
