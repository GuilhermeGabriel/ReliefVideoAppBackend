
## Description

Backend for [ReliefVideoAppFrontend](https://github.com/GuilhermeGabriel/ReliefVideoAppFrontEnd)

## Installation

```bash
# Versions used
#node v18.16.0
#npm 9.5.1

$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Technologies used: 

- NestJs
- NodeJs
- Prisma
- Sqlite
