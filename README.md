
## Description

Backend for [ReliefVideoAppFrontend](https://github.com/GuilhermeGabriel/ReliefVideoAppFrontEnd)

## Installation

```bash
# node v18.16.0
# npm 9.5.1

$ npm install
```

## Running the app

```bash
# start (http://localhost:8000/)
$ npm run start

# watch mode
$ npm run start:dev
```
Technologies used: 

- NestJs
- NodeJs
- Prisma
- Sqlite
